Module Hierarchy

  • IR.document: This package provides classes and modules neccessary for processing tokens and documents, including tokenization, reading files, and stemming
    • IR.document.PorterStemmer: Porter Stemming Algorithm This is the Porter stemming algorithm, ported to Python from the version coded up in ANSI C by the author.
    • IR.document.document: This module provides an interface to read documents and populate an object of type Document.
  • IR.indexer: This package provides classes and modules neccessary for indexing corpus
    • IR.indexer.constants
    • IR.indexer.indexer: This module defines the main Indexer class which builds an inverted file given a collection of documents.
  • IR.query_engine: This package provides classes and modules neccessary for loading partially loading an index and process and evaluate queries
    • IR.query_engine.OrderedSet
    • IR.query_engine.engine: Defines the class responsible for loading main index file into memory and perform queries on disk as well as memory
    • IR.query_engine.query: This module provides an interface to parse a query string and populate the query object
